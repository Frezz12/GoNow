# 001. Modular monolith

## Контекст

GoNow начинается с одного backend и небольшого числа продуктовых областей.

## Решение

Используем один Rust binary на Axum с модулями `auth`, `users`, `infrastructure` и `shared`.

## Причины

Это даёт общую транзакционную границу и простое локальное развёртывание без преждевременной распределённой сложности.

## Последствия

Модули должны общаться через явные модели и сервисы; новые области добавляются внутри приложения.

## Будущие изменения

Отдельный сервис допустим только при подтверждённой самостоятельной нагрузке или границе владения.
