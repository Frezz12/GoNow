# GoNow — design system v2

Единый визуальный контракт для iOS, Android и AI-агентов. GoNow — спокойный социальный сервис офлайн-активностей: нативный Apple-интерфейс, минимум декора, мягкая космическая атмосфера и стеклянные поверхности только там, где они помогают ориентироваться.

## Тема

Поддерживаются три режима: `system`, `light`, `dark`. Системный режим повторяет настройки устройства; ручной выбор применяется сразу и сохраняется. Нельзя делать экран, работающий только в светлом режиме.

## Бренд и поверхности

Фирменный градиент: `#7C5CFC → #B05CF5 → #F472B6`. Он допустим для основного CTA, выбранного состояния, progress и небольшого свечения — не для фона каждой карточки.

Liquid Glass строится из system material, мягкого tint, тонкого верхнего highlight, границы и очень лёгкой тени. Стекло применяется к панели навигации, контролам карты, небольшим overlay, карточкам и sheet. Длинные формы используют более плотную поверхность. При Reduce Transparency стекло заменяется читаемой непрозрачной surface-поверхностью.

## Токены

- Отступы: `4, 8, 12, 16, 20, 24, 32, 40`.
- Радиусы: `10, 14, 16, 22, 28, 30`; капсула — только для compact controls и CTA.
- Минимальная touch-цель: 44 pt / 48 dp.
- Системная типографика: SF Pro на iOS, системный sans-serif на Android. Поддерживаются Dynamic Type и font scaling.
- Анимации: 0.15, 0.25, 0.4 секунды; мягкий scale 0.98 при нажатии, без тяжёлого bounce и постоянного движения.

## Компоненты

- `GlassCard` — обычный информационный блок.
- `GlassPanel` — floating/overlay-панель.
- `PrimaryButton` — единственный основной CTA, высота 52–56, фирменный градиент, белый текст, loading/disabled/pressed.
- `SecondaryButton` — плотное стекло, accent-текст, без яркого градиента.
- `AppTextField` — видимая подпись, высота от 50, surface-фон, focus-border accent, inline error.
- `AppBadge` — небольшая категория или статус; цвет только в badge/иконке, не во всей карточке.
- `AppEmptyState` — SF Symbol, короткий текст, одно действие.

## Карта и навигация

Карта — главный визуальный экран. Поисковая строка, фильтры, геолокация и профиль располагаются поверх неё на плотном стекле и не закрывают карту. Точка пользователя — `locationAccent`; активность — `accentPrimary`/`accentSecondary`.

Нижняя панель содержит: Карта, Задания, Чат, Профиль. Панель плавающая, стеклянная, с отступом от краёв и Safe Area. Активная вкладка — `accentPrimary`. Кнопка «Создать» — отдельная приподнятая стеклянная капсула с брендовым градиентом, только на карте.

## Accessibility

- Контраст текста не ниже WCAG AA.
- Цвет никогда не единственный носитель смысла.
- У иконок действий есть VoiceOver label/hint.
- У форм — подписи, корректная клавиатура, ошибки рядом с полем и show/hide password.
- Учитываются Reduce Motion, Reduce Transparency, тёмная тема и увеличенный текст.

## Что нельзя делать

- Чистый чёрный фон, тяжёлые чёрные тени, агрессивный неон.
- Градиент на каждой карточке или поле.
- Цветы, декоративный шум, emoji в интерфейсных иконках.
- Вращающийся контур фокуса, бесконечные декоративные анимации и слишком прозрачные формы.
- Несогласованные наборы цветов, радиусов, иконок и отступов.

## Prompt для Android AI-агента

> Реализуй GoNow как спокойное нативное social-map приложение с режимами system/light/dark. Используй семантические токены из таблицы, SF/Material Symbols, мягкие glass-поверхности, фон `backgroundPrimary`, плотные формы и один градиентный CTA `#7C5CFC → #B05CF5 → #F472B6`. Нижняя панель плавающая: Карта, Задания, Чат, Профиль; «Создать» — отдельная приподнятая капсула на карте. Поддержи Dynamic Type, контраст, Reduce Motion и Reduce Transparency.
